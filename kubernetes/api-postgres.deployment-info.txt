NOTES:
** Please be patient while the chart is being deployed **

PostgreSQL can be accessed via port 5432 on the following DNS name from within your cluster:

    api-postgres.fp.svc.cluster.local - Read/Write connection
    api-postgres-read.fp.svc.cluster.local - Read only connection
To get the password for "api_daemon" run:

    export POSTGRES_PASSWORD=$(kubectl get secret --namespace fp api-postgres -o jsonpath="{.data.postgresql-password}" | base64 --decode)

To connect to your database run the following command:

    kubectl run api-postgres-client --rm --tty -i --restart='Never' --namespace fp --image docker.io/bitnami/postgresql:11.5.0-debian-9-r26 --env="PGPASSWORD=$POSTGRES_PASSWORD" --command -- psql --host api-postgres -U api_daemon -d api_daemon -p 5432



To connect to your database from outside the cluster execute the following commands:

    kubectl port-forward --namespace fp svc/api-postgres 5432:5432 &
    PGPASSWORD="$POSTGRES_PASSWORD" psql --host 127.0.0.1 -U api_daemon -d api_daemon -p 5432
